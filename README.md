# Flappy Bird Clone

A fun, enhanced Flappy Bird game built with Phaser 3.

## Features

- **Dynamic Difficulty**: Game gets harder as your score increases
- **Moving Pipes**: Pipes start moving vertically after score 8
- **Rocket Obstacles**: Rocket hazards appear after score 5
- **Sound Effects**: Web Audio API for game sounds (flap, score, collision)
- **Animated Graphics**: Smooth bird animations and visual effects
- **Best Score Tracking**: Local storage saves your best score
- **Responsive Design**: Works on desktop and mobile devices

## How to Play

1. Open `index.html` in a web browser
2. Tap/Click or press Space to make the bird flap
3. Navigate through the pipes without hitting them
4. Each pipe you pass increases your score
5. Try to beat your best score!

## Game Mechanics

- **Difficulty Scaling**: Pipe speed and frequency increase with score
- **Gap Variation**: Pipe gaps get smaller as you progress
- **Visual Patterns**: Pipes follow wave, alternating, and random patterns
- **Level Badge**: Shows your current difficulty level

## Files

- `index.html` - Game entry point
- `game.js` - Complete game logic and mechanics
- `style.css` - Styling and layout

## Technologies

- Phaser 3 (Game framework)
- HTML5 Canvas
- Web Audio API
- JavaScript (ES6+)
